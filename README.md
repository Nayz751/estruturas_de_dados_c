# Estruturas de Dados em C

Este repositório contém implementações de estruturas de dados utilizando a linguagem C, com foco em aprendizado de lógica, organização de dados e manipulação de memória.

## Conteúdos do projeto

- Pilha (implementação com lista encadeada)
- Fila
- Lista ordenada
- Lista não ordenada
- Calculadora utilizando estruturas
- Empilhamento de elementos

## Funcionalidades

- Inserção e remoção de elementos
- Impressão de pilhas e filas
- Classificação parcial de elementos
- Estruturas totalmente dinâmicas usando ponteiros

## Tecnologias

- Linguagem C
- Estruturas de dados: Pilha, Fila, Lista
- Manipulação de memória com `malloc` e `free`
